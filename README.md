# Weather, Exchange Rate and Notes Desktop App

A .NET desktop application that provides current weather information, global exchange rate data, and a simple notes management section in one interface.

## Overview

This project was developed as a visual programming desktop application. It allows users to search for the current weather of any city, view international currency exchange information, and create personal notes inside the application.

The project combines API usage, desktop UI design, and basic data management features.

## Features

- Search current weather information by city name
- Display weather details in the application interface
- View global currency exchange rate information
- Add and manage personal notes
- Simple and user-friendly desktop interface
- Built with .NET technologies

## Main Sections

### Weather Section

The weather section allows the user to enter a city name and retrieve current weather data for that city.

### Exchange Rate Section

The exchange rate section provides access to global currency exchange information.

### Notes Section

The notes section allows the user to add and manage personal notes inside the application.

## Technologies Used

- C#
- .NET
- Windows Forms / Desktop Application
- API Integration
- JSON Data Handling
